## Focus

### Simulating Physics in Gazebo

-   Rigid Body Dynamics.
-   Setting up a testing world with obstacles and terrain.

### High-fidelity rendering in Unity
-   Using Unity for photorealistic visualization.
-   Human-robot interaction scenarios.

### Simulating Sensors
-   **LiDAR**: Light Detection and Ranging for mapping.
-   **Depth Cameras**: Simulating RGB-D data (like Intel RealSense).
-   **IMU**: Inertial Measurement Units for balance and orientation.
